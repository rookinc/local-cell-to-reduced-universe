\section{The two readouts}

Let
\[
    L=\{O0,O1,B0,B1\}
\]
be the generated local answer cell.

The C readout orders \(L\) by ordinary shell first, branch shell second, and ascending rank inside each shell:
\[
    C(L)=[O0,O1,B0,B1].
\]

The anchor readout applies the Lift \& Twist order rule: the first branch state wraps to the front, ordinary ranks follow, and the remaining branch state stays terminal:
\[
    A(L)=[B0,O0,O1,B1].
\]

Thus both readouts contain the same hidden states.  They differ only by the visible order in which those states are read.
