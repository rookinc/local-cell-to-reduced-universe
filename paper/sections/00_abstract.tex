\begin{abstract}
Project 22 proved a local Lift \& Twist answer-cell theorem: inside the four-state reduced target, two state bits generate the C-side payload, anchor-side payload, overlap markers, and closed anchor node-path geometry.  This paper takes that local-cell theorem as a kernel and asks the next derivation question: does the reduced 16-candidate universe follow from the cell?

The answer proved here is conditional but exact.  Given the four generated local states \(O0,O1,B0,B1\), the C readout orders them as \(O0,O1,B0,B1\).  The anchor readout twists the same states as \(B0,O0,O1,B1\).  Their product generates the 16 visible candidates.  The hidden diagonal selects exactly candidates \([1,6,8,15]\), appearing visibly as the permutation \([1,2,0,3]=(0\ 1\ 2)(3)\).

This does not close Gap A.  It derives the reduced universe from the local-cell kernel, but it does not yet derive that kernel from native G60 provenance.
\end{abstract}
