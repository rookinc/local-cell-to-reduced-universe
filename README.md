# Local Cell to Reduced Universe

Project 23 in the Thalean graph theory workspace.

This project is a companion to Project 22, `22-lift-and-twist`.

Project 22 proved a local Lift & Twist answer-cell theorem: within the four-state reduced target, the state bits generate the C row, anchor column, selected candidate, C key, anchor key, C path, C values, anchor residue set, overlap markers, and closed anchor node path.

Project 23 asks the next derivation question:

Does the reduced 16-candidate universe follow from the local answer cell?

Working thesis:

    one local answer cell
    two readout orders
    sixteen visible pairings
    one hidden diagonal
    four selected candidates

Boundary:

This project does not claim full Gap A closure at creation. It aims to derive the reduced 16-candidate universe from the Project 22 local-cell kernel. It still does not derive that local-cell kernel from native G60 provenance.
